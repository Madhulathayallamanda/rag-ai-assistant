# RAG-Based AI Knowledge Assistant

## 🚀 Overview
This project implements a Retrieval-Augmented Generation (RAG) pipeline that enables context-aware question answering from documents using Large Language Models (LLMs).

## 🧠 Key Features
- Document ingestion (PDF)
- Text chunking and preprocessing
- Embedding generation
- FAISS-based vector search
- Context-aware response generation using LLM

## ⚙️ Tech Stack
- Python
- LangChain
- FAISS
- OpenAI API
- Sentence Transformers

## 🏗️ Architecture
1. Load documents
2. Split into chunks
3. Generate embeddings
4. Store in FAISS vector DB
5. Retrieve relevant chunks
6. Generate response using LLM

## 📌 Highlights
- Implements end-to-end RAG pipeline
- Optimized retrieval for semantic search
- Modular and scalable architecture

## ▶️ How it works
Run `app.py` to simulate a query-response pipeline.
